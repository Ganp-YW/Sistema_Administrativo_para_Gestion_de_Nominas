# Sistema Administrativo para Gestión de Nóminas

## Detalles Técnicos
Este proyecto es un sistema de escritorio desarrollado en **Java**.
- **Interfaz Gráfica:** Construida utilizando **JavaFX** y FXML, implementando estilos modernos mediante CSS.
- **Base de Datos:** Emplea **PostgreSQL** de forma embebida para garantizar la persistencia y facilidad de despliegue sin configuraciones externas complejas.
- **Arquitectura:** Sigue el patrón Modelo-Vista-Controlador (MVC) para mantener el código organizado, separando el diseño de la lógica de negocio y las consultas a base de datos (DAO).
